    • Block Drop

    Block Drop was a proof of concept, and was never meant to be released to the public. The game was proof that I could write my own code, and follow a specific design to the end. The models and animations were not mine and were taken from mixamo.com to see if I could implement animations. The game was created for mobile but an old PC version of the game can be downloaded here.

    Space Race

    Space Race was proof that I can improve upon my old code and model my own player models and obsticles. The game was heavily inspired by Block Drop and the mechinics are identical, however the code is new and bug free, unlike block drop. This was my frist use of the unity particle system, so my knowledge of the system was very limited. There is no PC version of this game, however the apk for android can be downloaded here.

    The Wasteland

    The Wasteland is a very ambitious project for myself, since I develop games on my own. The game is fallout 1 and 2 ispired, with the top-down isometric view and a ruined world theme. The foundation of the game is based on a youtube series made by Brackeys and Sebastian Lague, however many things were changed and added to make the game feel unique. This game is not finished and an ETA of release is unkown as a lot of work is still to be done. An early prototype of the game can be download here.
